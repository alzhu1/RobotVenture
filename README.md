# Robot Venture

### My first unity game. https://alzhu1.itch.io/robot-venture

![Demo Image](https://github.com/alzhu1/RobotVenture/blob/master/sample.png?raw=true)

### Controls
* Arrow Keys / WASD - Move
* Space - Jump/Wall Jump
* Z - Dash
* X - Climb
* C - Toggle Camera Mode
* M - Mute/unmute sound
* R - Restart Level
* Esc - Back to Title
